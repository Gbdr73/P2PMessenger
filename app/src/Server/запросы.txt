Формат запроса:

первые 3 символа - тип запроса:

reg - регистрация пользователя
lgn - авторизация пользователя
png - PING-запрос, для обновления IP пользователя
gip - получить IP пользователя

следующие 2 символа - длина имени пользователя (N)
следующие N символов - имя пользователя
следующие 64 символа только - хэш пароля пользователя

для запроса "gip" после этих полей следует длина имени пользователя IP которого запрашивается и его имя.

Формат ответа:

Первые 3 символа - код ответа

"000" - запрос выполнен успешно
"001" - неизвестный тип запроса или ошибка формата запроса
"002" - пользователь с таким именем уже зарегистрирован
"003" - неверный логин или пароль
"004" - пользователь не найден

для gip после кода ответа отправляется IP по запрошенному имени пользователя
